SAAi

Read the state correspondence of Assyria offline on iPhone and iPad

This software package was developed as part of my PhD research and builds on the extraordinarily rich work of Assyriologists and developers at the Open Richly Annotated Cuneiform Corpus, provided under generous open access and open source licenses. This app is currently in alpha state but source code is available on Github.
